# AI-Sentiment-Analyzer

Sentiment Analyzer

A Python-based Sentiment Analyzer that classifies text as Positive, Negative, or Neutral using Natural Language Processing (NLP).

Features

- Analyze user-provided text
- Detect Positive, Negative, and Neutral sentiments
- Simple and easy-to-use interface
- Built using Python

Technologies Used

- Python
- Natural Language Processing (NLP)
- Text Classification

Installation

Clone the repository:

git clone https://github.com/your-Princejaiswal18/Sentiment-Analyzer.git

Navigate to the project folder:

cd Sentiment-Analyzer

Install the required dependencies:

pip install -r requirements.txt

Usage

Run the program:

python main.py

Enter a sentence or paragraph, and the program will predict its sentiment.

Example

Input:

I really enjoyed this project. It was amazing!

Output:

Positive
